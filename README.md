## 如何使用

- 此 mod 适用于 Minecraft 1.21.4 Fabric.
- 这是客户端 mod.
- mod 依赖:
  - Fabric Language Kotlin
  - Mod Menu
  - Cloth Config

添加 mod 之后,  
使用 `/dglab connect` 打开连接二维码,  
然后使用 DG-LAB 3.0 app 扫描.
