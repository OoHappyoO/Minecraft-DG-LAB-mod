## How to use

- This mod is suitable for Minecraft 1.21.4 Fabric.
- It is client side.
- Dependent on:
  - Fabric Language Kotlin
  - Mod Menu
  - Cloth Config

After add the mod,  
Use `/dglab connect` to open the QRCode.  
Then scan it with DG-LAB 3.0 app.
